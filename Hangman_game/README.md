# 🎮 Hangman Game

This is a simple Hangman game built using Python and Flask.

## 🚀 Features

* A–Z letter buttons
* 6 lives system
* Win and Lose screen
* Shows correct word after game over
* Restart game option

## ▶️ Run Locally

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/
