# 🎯 CodeAlpha Internship Tasks

This repository contains two Python projects developed as part of the CodeAlpha Internship Program.

## 🤖 Rule-Based Chatbot

This project is a simple rule-based chatbot built using Python and Tkinter.  
It takes user input and responds with predefined replies.

🔗 [View Project](https://github.com/sweta331/codealpha_tasks/tree/main/rule-based-chatbot)

### 🚀 Features
- Keyword-based responses  
- Simple conversational flow  
- Beginner-friendly project  

### 🛠️ Technologies Used
- Python  
- Tkinter (GUI)  
- if-elif conditions  
- functions

## 🎮 Hangman Game (Flask Web App)

A web-based Hangman game built using Python and Flask.

🔗 [View Project](https://github.com/sweta331/codealpha_tasks/tree/main/Hangman_game)

### 🚀 Features
- Interactive A–Z buttons  
- 6 lives system  
- Win / Lose screen  
- Displays correct word after game over  
- Restart functionality  

### 🛠️ Tech Stack
- Python  
- Flask  
- HTML  
- CSS
